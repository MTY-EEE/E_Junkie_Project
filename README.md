# **E-JUNKIE SELENIUM TEST PROJESİ**

### 📝 **PROJE HAKKINDA**
Bu projede, Java Programlama dili ve Selenium Test yazılımı kullanılarak "E-Junkie Demo" alışveriş sitesinde otomasyon test işlemleri gerçekleştirilmiştir.

*E-Junkie Web Sitesi*
```
https://shopdemo.e-junkie.com/
```
*Alternatif Web Sitesi*
```
https://shopdemo.fatfreeshop.com/?
```
